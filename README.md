# 🚀 CPF Validator - Azure Function (.NET 8)

Projeto de validação de CPF utilizando:

- ✅ .NET 8 (Isolated Worker)
- ✅ Azure Functions v4
- ✅ Azure CLI
- ✅ Azure Functions Core Tools
- ✅ Visual Studio Code

---

## 📌 Descrição

API serverless para validação de CPF brasileiro utilizando algoritmo oficial de cálculo dos dígitos verificadores.

A função recebe um CPF via requisição HTTP POST e retorna se é válido ou não.

---

## 🛠️ Tecnologias Utilizadas

- .NET 8
- Azure Functions (Isolated)
- C#
- JSON
- GitHub

---

## ▶️ Como executar localmente

Pré-requisitos:

- .NET 8 SDK
- Azure Functions Core Tools

Execute: func start


A API ficará disponível em: http://localhost:7071/api/ValidateCpf
